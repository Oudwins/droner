Bruno endpoint configs

This folder contains per-endpoint configuration files generated for the server endpoints
in this repository. Each file is a small YAML describing method, path and a brief
description. These are intended as a lightweight starting point for integration or
automated testing tooling named "bruno" used by the team.

Files:
- version.yml
- shutdown.yml
- sessions.list.yml
- sessions.create.yml
- sessions.delete.yml
- sessions.complete.yml
- sessions.nuke.yml
